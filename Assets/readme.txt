3D Asteroids style game

Default Controls
---------------------------------------
Keyboard:
WASD keys  ------------  Move Ship
Spacebar  -------------  Thrust
Left CTRL  ------------  Fire
Mouse  ----------------  Rotate View


Gamepad:
Left Joystick  --------  Move Ship
Right Joystick  -------  Rotate View
Left Trigger  ---------  Thrust
R Button  -------------  Fire
